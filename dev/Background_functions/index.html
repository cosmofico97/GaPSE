GaPSE.z_eff — Constant
 const z_eff :: Float64

The effective redshift $z_\mathrm{eff}$ is calcuated as follows:

\[\begin{align*} z_\mathrm{eff} := \frac{ \int \mathrm{d}^3\mathbf{s} \, \phi^2(\mathbf{s}) \, z(s) }{ \int \mathrm{d}^3\mathbf{s}\, \phi^2(\mathbf{s}) } &= \frac{ \int_0^\infty \mathrm{d}s \, s^2 \, \phi^2(s) \, z(s) \times \int_{4\pi}\mathrm{d}^2\hat{\mathbf{s}} \, W^2(\hat{\mathbf{s}}) }{ \int_0^\infty \mathrm{d}s \, s^2 \, \phi^2(s)\times \int_{4\pi}\mathrm{d}^2\hat{\mathbf{s}} \, W^2(\hat{\mathbf{s}}) } \\[5pt] &= \frac{ \int_0^\infty \mathrm{d}s \, s^2 \, \phi^2(s) \, z(s) }{ \int_0^\infty \mathrm{d}s \, s^2 \, \phi^2(s) } \\[4pt] &= \frac{3}{s_\mathrm{max}^3 - s_\mathrm{min}^3} \, \int_{s_\mathrm{min}}^{s_\mathrm{max}} \mathrm{d}s \, s^2 \, z(s) \end{align*}\]

where we have used our assuption on separability of the window function

\[ \phi(\mathbf{s}) = \phi(s) \, W(\hat{s})\]

and their definitions.

See also: ϕ, W
